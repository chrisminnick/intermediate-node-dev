# Intermediate Node.js Development - 5-Day Course Schedule

## Course Overview

This intensive 5-day course builds on fundamental Node.js knowledge to create production-ready applications with advanced patterns, security, and scalability considerations.

## Daily Schedule Template

- **09:00-10:30** - Morning Lecture Session 1
- **10:30-10:45** - Break
- **10:45-12:00** - Morning Lecture Session 2
- **12:00-13:00** - Lunch Break
- **13:00-14:30** - Hands-on Lab Session 1
- **14:30-14:45** - Break
- **14:45-16:00** - Hands-on Lab Session 2
- **16:00-16:30** - Q&A and Day Wrap-up

---

## Day 1: Advanced Asynchronous Programming & Express Design

**Focus**: Event loop mastery, async patterns, and scalable API architecture

### Morning Sessions

- **Session 1**: Event Loop Deep Dive & Performance Considerations
- **Session 2**: Async Iterators, Generators & Worker Threads

### Afternoon Labs

- **Lab 1**: Building Async Iterators for Data Processing
- **Lab 2**: Worker Thread Implementation for CPU-Intensive Tasks

---

### Afternoon Labs

- **Lab 2**: Advanced MongoDB Operations with Transactions

## Prerequisites Verification

Before starting, ensure students can:

- Build basic Express servers
- Understand JavaScript promises and async/await
- Work with npm and package.json
- Basic familiarity with databases (MongoDB or SQL)
- Comfortable with command line operations

## Required Software

- Node.js 18+
- MongoDB (local or cloud)
- Redis
- Docker Desktop
- Visual Studio Code or preferred IDE
- Postman or similar API testing tool
